## 哈夫曼树的概念

> 示例：将学生百分制成绩转换为五分制成绩，<60 => E/60-79 => D/70-79 => C/80-89 => B/90-100 => A

```c
if(score < 60) 
  grade = 'E'
else if(score < 70)
  grade = 'D'
else if(score < 80)
  grade = 'C'
else if(score < 90)
  grade = 'B'
else if(score < 100)
  grade = 'A'
  
// 这种用于描述分类过程的结构可以改为二叉树，称为判断树
```
